Пример запуск скрипта:
---
/bin/python3 task2.py -s task2_source -t task2_target -i 5 -l sync.log

shutil используется ддля копирования файлов
hashlib используется для проверки md5 суммы файла, что обновить если файл был изменен

Скрипт обходит директорию источник для копировния/создания объектов в целейвой директории
После обхожит целевую аудиторию для удаления объектов которых нет в источнике
